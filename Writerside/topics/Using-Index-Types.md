# Using Index Types

A classe `Collection<T>` restringe os tipos isso pode ser aceitavel usando um shape type, que faz com que todos os objetos é lidado tendo a propriedade `name` que pode esr usado como key para armazenar e retornar objetos no Map

TS prove um conjunto de features relacionadas que permite qualquer propriedade podendo um objeto ser usado como uma key enquanto preserva uma segurança de tipo (type safety)
